# Phobos Frontend

Frontend application for Phobos, built with React, TypeScript, and Vite.

## Backend Repository

The backend repository can be found at: [https://github.com/puffious/phobos](https://github.com/puffious/phobos)

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- Firebase
- React Router
- React Query

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or bun package manager

### Installation

```bash
# Install dependencies
npm install
# or
bun install
```

### Development

```bash
# Run development server
npm run dev
# or
bun run dev
```

The app will be available at `http://localhost:8080`

### Build

```bash
# Production build
npm run build
# or
bun run build
```

### Preview

```bash
# Preview production build
npm run preview
# or
bun run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── hooks/          # Custom React hooks
├── lib/            # Utilities and configurations
└── pages/          # Page components
```
